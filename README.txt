Quick use:
1. Open index.html directly for normal browser use.
2. For full PWA installation, the folder must be served over HTTPS or localhost.
3. Upload the entire folder to a static host such as GitHub Pages, Netlify, Cloudflare Pages, or another HTTPS web host.
4. Open the hosted address in Chrome or Samsung Internet and choose Install App / Add to Home screen.

The app stores the default risk and evaluation settings locally on each device.
Commission is set to 0.04% per side.
