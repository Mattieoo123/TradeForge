Files:

- service-worker.js
- icon.svg

Quick use:
1. Open index.html directly for normal browser use.
2. For full PWA installation, serve this folder over HTTPS or localhost.
3. Upload all files to the root of a GitHub Pages repository.
4. Open the hosted address in Chrome or Samsung Internet and choose Install App or Add to Home screen.

The app stores the default risk and evaluation settings locally on each device.
Commission is set to 0.04% per side.
